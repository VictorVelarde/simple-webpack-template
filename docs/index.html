Simple Webpack template

1. What is this?

This is a little web template. It provides a quick start for a basic project with Webpack + some other goodies working out of the box:

  • ES6, thx to Babel
  • SASS styles with css-reset
  • Hot Module Replacement (HMR) for both of them

2. How do I use it?

  • To Debug your project...

                            npm start

    It will start a local dev server at :8080, loading index.html in HMR mode

  • To Build your project

                        npm run build

    It will create js and css minified bundles at `docs`, with their corresponding .map files
