simple webpack template

1. What is this?

This is a little web template. It provides a quick start for a basic project using Webpack + some other goodies working out of the box:

  • ES6, thx to Babel
  • SASS styles with css-reset
  • Hot Module Replacement (HMR) for both JS & CS

2. How do I use it?

  • To DOWNLOAD

    git clone https://github.com/VictorVelarde/simple-webpack-template.git

    - It will clone the repo locally

  • To PREPARE

    cd simple-webpack-template && npm install

    - It will download its dependencies

  • To DEBUG your project:

    npm start

    - It will start a local dev server at :8080, loading index.html in HMR mode

  • To BUILD your project

    npm run build

    - It will create js and css minified bundles at `docs`, with their corresponding .map files
